<h3>3. Должностная инструкция инженера по качеству</h3><h2>Описание должности по данным портала obzorzarplat.ru</h2>
<table class="job_description"><tr><td>Название должности: </td><td>Инженер по качеству</td></tr><tr><td>Другое название должности: </td><td>Инженер ОТК, специалист по качеству</td></tr><tr><td>Подчиняется: </td><td>Начальнику отдела </td></tr><tr><td>Подчинённые: </td><td>Отсутствуют</td></tr><tr><td>Цель: </td><td>Контроль качества выпускаемой продукции/выполняемых работ (услуг).</td></tr><tr><td>Задачи: </td><td>Обеспечение выполнения заданий по повышению качества выпускаемой продукции, выполняемых работ (услуг), осуществление контроля над деятельностью подразделений предприятия по обеспечению соответствия продукции, работ (услуг) современному уровню развития науки и техники, требованиям потребителей на внутреннем рынке, а также экспортным требованиям и др.</td></tr><tr><td>Функции: </td><td><li>Выполняет работу по разработке предложений и планированию работ по повышению качества услуг связи, осуществляет контроль за деятельностью всех подразделений предприятия по обеспечению соответствия работ и услуг действующим нормам и стандартам; </li>
  <li>Координирует работу по разработке, совершенствованию и внедрению системы качества, созданию стандартов предприятия, контролирует их соблюдение; </li>
  <li>Координирует разработку и контроль выполнения целевых программ и задач по управлению качеством; </li>
  <li>Участвует в разработке анкет и методик для сбора и анализа информации; </li>
  <li>Производит сбор и анализирует информацию, необходимую для выполнения задач по управлению качеством; </li>
  <li>Участвует в разработке методик и инструкций по текущему контролю качества услуг и работ; </li>
  <li>Осуществляет контроль за качеством выполнения работ, прямо или косвенно влияющих на качество предоставления услуг, за деятельностью подразделений предприятия по обеспечению соответствия услуг современному уровню развития науки и техники, требованиям потребителей на внутреннем рынке; </li>
  <li>Анализирует информацию, полученную на различных этапах предоставления услуг, показатели качества, характеризующие предоставляемые услуги, и принимает меры по предотвращению предоставления услуг, не соответствующих установленным требованиям; </li>
  <li>Рассматривает и анализирует рекламации и претензии к качеству услуг, готовит заключения и ведет переписку по результатам их рассмотрения; </li>
  <li>Изучает причины, вызывающие ухудшение качества услуг, брака, участвует в разработке и внедрению мероприятий по их устранению; </li>
  <li>Изучает передовой отечественный и зарубежный опыт по разработке и внедрению систем управления качеством; </li>
  <li>Ведет учет и составлять отчетность о деятельности предприятия по управлению качеством, предоставляет отчетность в вышестоящую организацию в установленные сроки; </li>
  <li>Осуществляет контроль и оказывать методическую помощь подразделениям цеха при разработке структуры и организации процессов, оформлении документов системы качества; </li>
  <li>Осуществляет подготовку и координацию работ для проведения самооценки деятельности предприятия на соответствие критериям в области качества предоставления услуг связи; </li>
  <li>Составляет отчет о результатах самооценки и предоставляет его в вышестоящую организацию в установленные сроки; </li>
  <li>Осуществляет работу по подготовке и проведению внутренних проверок качества на соответствие их требованиям стандартов серии ИСО 9000; </li>
  <li>Выполняет Правила внутреннего трудового распорядка, Инструкции по охране труда, по пожарной безопасности, требования охраны труда, техники безопасности и пожарной безопасности, предъявляемые к производственным помещениям; </li>
  <li>Организует на предприятии работы по разработке, совершенствованию и внедрению системы качества, по созданию стандартов предприятия и контролю за их соблюдением.</li>

</td></tr><tr><td>Требования к опыту и квалификации: </td><td>Инженер по качеству относится к категории специалистов;
<p>На должность инженера по качеству назначается лицо, имеющее высшее образование и стаж работы по специальности от 1 года;
<p>Должен знать:
  <li>постановления, распоряжения, приказы, методические и нормативные материалы по вопросам управления качеством; </li>
  <li>политику руководства в области качества; </li>
  <li>стандарты и документированные процедуры системы менеджмента качества, действующие на предприятии; </li>
  <li>производственные мощности предприятия; </li>
  <li>виды предоставляемых услуг, тарифы на услуги; </li>
  <li>организацию производства; </li>
  <li>специализацию цехов, участков, производственные связи между ними; организационно-функциональную структуру предприятия.</td></tr></table>
