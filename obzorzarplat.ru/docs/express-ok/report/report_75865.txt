<h3>3. Должностная инструкция Бухгалтера по учету себестоимости</h3><h2>Описание должности по данным портала obzorzarplat.ru</h2>
<table class="job_description"><tr><td>Название должности: </td><td>Бухгалтер по учету себестоимости</td></tr><tr><td>Другое название должности: </td><td>Бухгалтер по учету материалов</td></tr><tr><td>Подчиняется: </td><td>Главному бухгалтеру</td></tr><tr><td>Подчинённые: </td><td>Отсутствуют</td></tr><tr><td>Цель: </td><td>Осуществление бухгалтерского учета себестоимости материалов.</td></tr><tr><td>Задачи: </td><td>Ведение бухгалтерского учета в соответствии с требованиями действующего законодательства в части, касающейся учета принадлежащих организации материалов, оприходования материалов, расчета фактической себестоимости материалов и т.п.</td></tr><tr><td>Функции: </td><td><li>Участвует в разработке и осуществлении мероприятий, направленных на соблюдение финансовой дисциплины и рациональное использование ресурсов;</li>
<li>Осуществляет прием и контроль первичной документации на данном участке бухгалтерского учета и подготавливает их к счетной обработке;</li>
<li>Отражает на счетах бухгалтерского учета операции по учету материалов;</li>
<li>Составляет отчетные калькуляции по фактической себестоимости материалов с учетом всех затрат;</li>
<li>Производит начисление налогов, возникающих на данном участке;</li>
<li>Обеспечивает руководителей, кредиторов, инвесторов, аудиторов и других пользователей бухгалтерской отчетности сопоставимой и достоверной бухгалтерской информацией по соответствующему участку учета;</li>
<li>Разрабатывает рабочий план счетов, формы первичных документов, применяемые для оформления хозяйственных операций, по которым не предусмотрены типовые формы, а также формы документов для внутренней бухгалтерской отчетности, участвует в определении содержания основных приемов и методов ведения учета и технологии обработки бухгалтерской информации;</li>
<li>Подготавливает данные по соответствующему участку бухгалтерского учета для составления отчетности, следит за сохранностью бухгалтерских документов, оформляет их в соответствии с установленным порядком для передачи в архив;</li>
<li>Выполняет работы по формированию, ведению и хранению базы данных бухгалтерской информации, вносит изменения в справочную и нормативную информацию, используемую при обработке данных.</li></td></tr><tr><td>Требования к опыту и квалификации: </td><td>Бухгалтер по учету себестоимости относится к категории специалистов;
<p>На должность бухгалтера по учету себестоимости назначается лицо, имеющее среднее профессиональное (экономическое) образование без предъявления требований к стажу работы или специальную подготовку по установленной программе и стаж работы по учету и контролю не менее 3 лет.</td></tr></table>
