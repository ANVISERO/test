<li>Моделирует ситуации, которые могут возникнуть в условиях эксплуатации программного обеспечения;
<li>Выполняет тестирование программных продуктов;
<li>Выполняет нагрузочные тестирования;
<li>Составляет документацию для проведения функционального тестирования;
<li>Участвует в проведении опытных эксплуатаций программных продуктов;
<li>Заполняет таблицы баз данных тестовыми данными;
<li>Анализирует результаты, полученные во время прохождения тестов;
<li>Классифицирует выявленные ошибки и заносит их в базу данных для текущего программного продукта;
<li>Контролирует процесс ликвидации выявленных ошибок разработчиком ПО;
<li>Общается с разработчиками;
<li>Консультирует клиентов;
<li>Работает в связке с разработчиком;
<li>Создает тест-планы, тест-кейсы.</td></tr><tr><td>Требования к опыту и квалификации: </td><td>Тестировщик относится к категории специалистов;

<p>Должен знать:
<li>основные технологии построения ПО и структуры программных комплексов;
<li>знание операционных систем семейства Windows на уровне продвинутого пользователя;
<li>язык запросов SQL;
<li>скриптовые языки;
<li>принципы программирования;
<li>специальное ПО для автоматизированного тестирования и регистрации ошибок (WinRunner, TestComplete, TestExecute, TestRecorder);
<li>английский язык (как минимум — на уровне чтения технической документации);
<li>принципы создания тест-кейсов.</td></tr></table>
