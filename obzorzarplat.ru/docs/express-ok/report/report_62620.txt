<h3>3. Должностная инструкция начальника управления проектами</h3><h2>Описание должности по данным портала obzorzarplat.ru</h2>
<table class="job_description"><tr><td>Название должности: </td><td>Начальник управления проектами</td></tr><tr><td>Другое название должности: </td><td>Менеджер проекта/ Руководитель проекта</td></tr><tr><td>Подчиняется: </td><td>Генеральному директору</td></tr><tr><td>Подчинённые: </td><td>Руководители проектов, менеджеры проекта </td></tr><tr><td>Цель: </td><td>Достижение конечных целей и задач от внедрения новых проектов.</td></tr><tr><td>Задачи: </td><td>Осуществление проекта, целей и конечных результатов. Разработка концепции, планов, координация деятельности сотрудников.</td></tr><tr><td>Функции: </td><td><ul><li>1.  Определяет устав, цели, задачи и результат проекта. </li>
<li>2.  Продумывает и составляет план по  подготовке и внедрению нового проекта, определяет контрольные точки. </li>
<li>2.1.  Определяет состав работ, необходимых для проработки и внедрения проекта. </li>
<li>2.2.  Определяет и документирует зависимости между работами. </li>
<li>2.3.  Оценивает продолжительность работ, составляет критический путь. </li>
<li>2.4.  Определяет количество времени необходимое для осуществления проекта. </li>
<li>3.  Определяет количество и оценивает стоимость ресурсов, требуемых для выполнения работ проекта. </li>
<li>4.  Оценивает стоимость и определяет бюджет проекта. </li>
<li>5.  Выбирает команду проекта. </li>
<li>5.1.  Определяет профессиональные навыки, необходимые участникам команды проекта. </li>
<li>5.2.  Определяет ответственные Управления и ответственные лица, которые будут участвовать в процессах проработки и внедрения проекта. </li>
<li>5.3.  Прописывает цепочку взаимосвязей между участниками команды проекта. </li>
<li>5.4.  Продумывает систему мотивации команды проекта. </li>
<li>6.  Организует собрания команды проекта. </li>
<li>7.  Принимает участие в разработке детального бизнес-плана. </li>
<li>8.  Контролирует подготовку необходимой документации для внедрения нового проекта. </li>
<li>9.  Контролирует подготовку необходимой документации для внедрения нового проекта. </li>
<li>9.1.  Обеспечивает своевременные сбор, накопление,  распространение, хранение и последующее использование информации проекта. </li>
<li>9.2.  Координирует предоставление нужной информации в обусловленные сроки всем участникам проекта. </li>
<li>10.  Контролирует и отслеживает мероприятия по проработке и внедрению и при необходимости корректирует данные процессы. </li>
<li>10.1.  Контролирует соблюдение сроков процессов проработки и внедрения. </li>
<li>10.2.  Контролирует изменения бюджета проекта</li>
<li>10.3.  Отслеживает отклонения от плана, вносит корректировки в план и согласует его со всеми участниками проекта. </li>
<li>10.4.  Анализирует возможное влияние отклонений в выполненных объемах работ на ход реализации проекта в целом. </li></ul>
</td></tr><tr><td>Требования к опыту и квалификации: </td><td><ul><li>Высшее экономическое образование</li>
<li>Специальное обучение, допуски: знание MS Project или MS Central</li>
<li>Навыки: работы на ПК (пакет MS Office, Internet) </li>
<li>Опыт работы: на руководящей должности не менее 3-х лет</li>
<li>Профессиональные знания: </li>
<li>Принципы обработки информации; </li>
<li>Стандарты и методологии проектного подхода; </li>
<li>Управление временем; </li>
<li>Навыки ведения переговоров; </li>
<li>Менеджмент. </li></ul>
</td></tr></table>
