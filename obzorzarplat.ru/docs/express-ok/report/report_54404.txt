<h3>3. Должностная инструкция менеджера по подготовке производства</h3><h2>Описание должности по данным портала obzorzarplat.ru</h2>
<table class="job_description"><tr><td>Название должности: </td><td>Менеджер по подготовке производства</td></tr><tr><td>Другое название должности: </td><td>Менеджер по планированию производства / Инженер по подготовке производства </td></tr><tr><td>Подчиняется: </td><td>Генеральному директору / Начальнику производства</td></tr><tr><td>Подчинённые: </td><td>Специалист по планированию</td></tr><tr><td>Цель: </td><td>Планирует и анализирует производственные показатели для повышения эффективности.</td></tr><tr><td>Функции: </td><td><ul>
<li>Отвечает за составление производственных планов;
<li>Анализирует показатели производственной деятельности и готовит отчеты для руководства компании по результатам и причинам отклонений показателей от плановых;
<li>Готовит предложения по повышению эффективности производства, по оптимальному использованию рабочего времени производственного персонала, а также использованию материалов и оборудования;
<li>Участвует в разработке инновационных методов работы и технологических цепочек, и предложений  по внедрению их на производстве;
<li>Может участвовать в работе по выявлению возможных рисков и внедрению на производстве инновационных процедур и техники безопасности;
<li>Может быть включен в группу по мониторингу затрат на строительные проекты;
</ul>
</td></tr><tr><td>Требования к опыту и квалификации: </td><td><ul>
<li>Высшее техническое образование, желательно дополнительно в области экономики;
<li>Опыт работы не менее 5 лет;
<li>Знание иностранного языка;
<li>Опытный пользователь ПК (MS Project; Project Expert, AutoCAD);
<li>Знание методологии управления проектами PMI, PHBOOK;
<li>Коммуникабельность, презентабельный внешний вид.
</ul>
</td></tr></table>
