В рамках исследовательского проекта “<strong>Компенсация труда в российских компаниях</strong>” расширено представление структуры отдела персонала, сейчас оно представлено восемью должностями: <em>Директор по персоналу, Начальник отдела персонала, Начальник отдела кадров, Менеджер по персоналу, Менеджер по подбору персонала, Менеджер по обучению, Менеджер по кадровому делопроизводству, Инженер по охране труда</em>. В планах расширить должностную матрицу подразделения по работе с персоналом до 18 должностей.
