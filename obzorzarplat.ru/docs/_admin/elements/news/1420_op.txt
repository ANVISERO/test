В рамках исследовательского проекта “Компенсация труда в российских компаниях” расширено представление структуры отдела персонала, сейчас оно представлено восемью должностями: Директор по персоналу, Начальник отдела персонала, Начальник отдела кадров, Менеджер по персоналу, Менеджер по подбору персонала, Менеджер по обучению, Менеджер по кадровому делопроизводству, Инженер по охране труда. В планах расширить должностную матрицу подразделения по работе с персоналом до 18 должностей.
