<p><b>9 февраля 2011</b> года пройдет очередное заседание клуба "Кочубей" на тему: "Как не стать кузницей кадров для конкурентов. Удержание талантливых сотрудников."</p>
<p>Сотрудники - один из основных ресурсов любой компании. Уход ключевого специалиста, менеджера по продажам, работавшего с ключевыми клиентами, топ-менеджера могут стать для компании не просто сильным ударом, а началом конца. Именно поэтому одна из основных задач любого директора по персоналу - создать условия для удержания талантливых сотрудников.</p>
<p>На заседании будут обсуждаться методы и инструменты, с помощью которых Вы можете снизить риск ухода ключевых специалистов и руководителей и максимально затруднить работу рекрутерам конкурентов.</p>
